# Event Schema Stable Names

Use these event names in code:

- lesson_started
- concept_seen
- quiz_attempted
- quiz_correct
- quiz_wrong
- hint_used
- review_completed
- code_attempted
- code_passed
- skill_mastered
- skill_regressed

Common fields:

- id
- user_id
- skill_id
- learning_item_id
- review_card_id
- event_type
- event_time
- metadata
- created_at
